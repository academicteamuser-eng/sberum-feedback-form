# sberum-feedback-form
# Форма обратной связи для платформы СберУм

Веб-форма для сбора обратной связи от пользователей платформы СберУм.

## Функционал

- Форма сбора обратной связи
- Условное отображение полей в зависимости от типа (баг/фича)
- Загрузка файлов и вставка ссылок
- Валидация обязательных полей
- Отправка данных в Google Sheets
- Модальные окна подтверждения/ошибки
- Защита от потери данных при закрытии страницы

## Установка

1. Клонируйте репозиторий или скачайте файлы
2. Разместите файлы на веб-сервере или хостинге
3. Настройте Google Apps Script для работы с Google Sheets

## Настройка Google Sheets

### Шаг 1: Создайте Google Apps Script
1. Откройте вашу Google Таблицу
2. Нажмите "Расширения" → "Apps Script"
3. Удалите весь код и вставьте код из `google-apps-script.js`

### Шаг 2: Разверните как веб-приложение
1. В редакторе Apps Script нажмите "Развернуть" → "Новый вид развертывания"
2. Выберите "Веб-приложение"
3. Настройте доступ: "Доступ: любой, у кого есть ссылка"
4. Нажмите "Развернуть"
5. Скопируйте URL веб-приложения

### Шаг 3: Обновите конфигурацию
В файле `script.js` замените `WEB_APP_URL` на URL вашего веб-приложения.
